#概要
現在時刻に応じて、挨拶の内容を下記のようにそれぞれ返す機能を作成する。  
*5:00~11:59の場合、「おはようございます」と返す
*12:00~17:59の場合、「こんにちは」と返す
*18:00~4:59の場合、「こんばんは」と返す  

例: 13時にgreeter.greet()を呼ぶと”こんにちは”と返す  
  
余裕があれば場所に応じた挨拶もできるようにする。
